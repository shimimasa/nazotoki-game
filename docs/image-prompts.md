## 生成時の注意

| 項目 | 指定 |
|---|---|
| **立ち絵サイズ** | 768×1024px 以上推奨。webpで保存 |
| **背景サイズ** | 1920×1080px（16:9）推奨。webpで保存 |
| **背景の白除去** | 立ち絵は白背景で生成後、背景除去ツール（remove.bg等）で透過PNGに→webp変換 |
| **スタイル統一** | 全画像を同じツール・同じシード値系で生成すると統一感が出る |
| **NG** | リアル調、3D、実写風は避ける。アニメ/イラスト調で統一 |
