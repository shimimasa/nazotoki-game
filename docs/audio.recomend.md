  ダウンロードについて

  自動ダウンロードは不可です。理由:
  - 効果音ラボ: 直リンクDL禁止（サイトポリシー）
  - 魔王魂/DOVA-SYNDROME: ダウンロードボタン経由が必要

  人間タスク（所要10-15分）:
  1. https://soundeffect-lab.info/sound/button/ から SE 3個DL
  2. https://soundeffect-lab.info/sound/anime/ から SE 4個DL
  3. https://maou.audio/tag/%E6%8E%A8%E7%90%86/ から BGM 3曲DL
  4. DLしたファイルを game/public/audio/bgm/ と game/public/audio/se/ にリネーム配置

  配置後、AudioManagerは既存のパス規約（/audio/bgm/{name}.mp3, /audio/se/{name}.mp3）で自動的に読み込めます。
