## 生成手順

### Gemini Imagen の場合

1. Google AI Studio (aistudio.google.com) を開く
2. 共通設定プレフィックスを入力
3. 各キャラ×表情のプロンプトを追記して生成
4. 透過背景のPNGをダウンロード
5. `cwebp -q 85 input.png -o output.webp` で変換

### Midjourney の場合

1. 共通設定を `--style raw --ar 2:3` と組み合わせ
2. `--no background` で透過背景を指定
3. PNG出力 → webp変換

### DALL-E (ChatGPT) の場合

1. 共通設定 + 各プロンプトを入力
2. 「transparent background, PNG format」を必ず追記
3. ダウンロード → webp変換

### 一括webp変換コマンド

```bash
# ImageMagick が必要
for f in *.png; do
  convert "$f" -quality 85 "${f%.png}.webp"
done

# または cwebp（libwebpが必要）
for f in *.png; do
  cwebp -q 85 "$f" -o "${f%.png}.webp"
done
```

### 配置先

```bash
# game/public/images/sprites/ に配置
mv *.webp game/public/images/sprites/
```

## 画風統一のコツ

- **同じセッション内で全キャラを生成**する（画風のブレを防ぐ）
- **1キャラの6表情を連続生成**してから次のキャラに進む
- 最初にboy-a normalを生成し、その画像を参照画像として残りを生成
- 制服（白シャツ+紺ベスト）を全キャラで統一することで一体感を出す
