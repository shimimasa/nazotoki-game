# 図工探偵団 変換ノート

## craft-01（図工探偵団 Vol.1）

**変換日**: 2026-03-29 / シーン10 / キャラ5 / 選択肢1問 / ヒナタ→boy-a, ツクル→boy-b, カナデ→girl-b, スミレ→girl-c, 山田先生→teacher-f

### 気づき
- **証拠カードが情報量多め**: 色カード一覧・行動記録・パレット観察・使用記録簿の4枚をナレーションのみで構成。キャラの出番がないため単調にならないよう、各カードに「考えてみよう」的な問いかけナレーションを末尾に追加
- **混色テーマの視覚化限界**: ゲーム上ではパレットの色を実際に見せられないため、テキストで「5段階のオレンジ」と記述。実際の授業ではパレット実物を見せることで補完可能
- **犯人当て型の典型構造**: 消去法パターン（3人のパレットにオレンジなし→残り1人）は証拠カード3で明確に提示。Card5（先生の目撃証言）でアリバイ崩壊の決定打
- **感情アーク**: ヘッドライン型のためプロローグの衝撃を重視。SE(crash)+BGM切り替えで緊張感を演出

## craft-05（図工探偵団 Vol.5）

**変換日**: 2026-03-29 / シーン11 / キャラ5 / 選択肢1問 / 同一マッピング

### 気づき
- **前提崩壊型（犯人＝被害者本人）**: 「外部犯人がいたずらした」という全員の思い込みを崩す構造。選択肢に「ヒナタ自身」を入れるのが鍵。ゲームではヒナタ以外のプレイヤーがいないため、choice feedbackで前提崩壊の驚きを演出
- **追加ハンドアウト対応**: 原作の「色彩理論カード」を独立シーン(color-theory)として挿入。evidence-2とevidence-3の間に配置し、補色の知識を先に提示してから3層構造の分析に進む流れ
- **11シーン構成**: 追加ハンドアウト分で通常より1シーン多い。テンポが間延びしないよう、color-theoryシーンはナレーションのみで簡潔に
- **deductionの証言が3人**: スミレ（3層分析）、カナデ（表情分析）、ツクル（つぶやき証言）の3人が順に情報を出す。他シリーズより証言が多いが、前提崩壊への段階的誘導として必要
