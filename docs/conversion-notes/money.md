## money-01（お金の探偵団 Vol.1）

**変換日**: 2026-03-23

### サマリー
- シーン数: 16 / キャラ数: 5 / 選択肢: 2問
- 型: 犯人特定（消去法 + 帳簿計算）
- スプライト: yuta→boy-b, sakura→girl-a, kaito→boy-d, nanoha→girl-c, sensei→teacher-m

### 気づき
- **「ズレの質」が推理の核心**: 4人中3人に帳簿のズレがあるが、10円（書き間違い）と120円（意図的な未記録）では質が全く異なる。VNでは「10円のズレ→レシートで説明つく」をサクッと提示してから「120円のズレ→レシートがない！」の衝撃を出す、二段階構成が効果的。
- **「なぜ記録しなかったか」の選択肢**: 犯人当て+動機推理の2問構成。「忘れた」「禁止だから隠した」「金額が小さいから」等の選択肢で、プレイヤーに金融教育の本質を考えさせる。
- **全員のズレを修正するエピローグ**: ユウタとナノハも自分の帳簿を修正する。カイトだけでなく全員が学ぶ構造。「水曜：にこにこ堂 お菓子 120円」という1行の追加で物語が完結する演出。
- **programming-01と同じ「全員に落ち度」パターン**: ユウタ（覗き見）、サクラ（見て見ぬふり）、ナノハ（内緒にした）。教育シナリオの共通パターンとして定着。

### 課題
- **先生キャラが男性（teacher-m）に固定されがち**: science-01, digital-01, money-01で全てteacher-m。programming-01だけteacher-f。女性教師の出番を増やすか、シナリオに応じて使い分けるルールが欲しい。
- **背景は教室1枚で足りる**: 駄菓子屋シーンもナレーションで処理。特殊背景は不要。
- **素体の重複**: boy-b(情報型), boy-d(行動型), girl-a(感情型), girl-c(観察型)のパターンがdigital-01と完全に同じ。シリーズごとに同じ素体の組み合わせになるのは、キャラの役割分担が同じテンプレートに基づくため。4人構成のVNでは避けられない。

## money-02（お金の探偵団 Vol.2）

**変換日**: 2026-03-24 / シーン12 / キャラ5 / 選択肢3問 / yuta→boy-b, sakura→girl-a, kaito→boy-d, nanoha→girl-c, sensei→teacher-m

### 気づき
- **おつり計算がVNの段階表示で学習体験になる**: 4人の「500円−値段＝おつり」を1人ずつnarrationで表示し、ナノハだけ❌を出す。プレイヤーが自分で計算して「合わない！」と気づける。金融教育とVNテンポが合致。
- **硬貨の内訳比較が推理の核心**: お店が渡した10円玉3枚 vs ナノハの財布の10円玉4枚。この「1枚の差」をユウタのセリフで段階的に出すと、「元からあったんだ！」の発見が生まれる。Vol.1の帳簿ズレと同じ「数値の差分」構造だがアプローチが異なる。
- **犯人の動機が「誤解」**: ナノハは嘘をつく意図がなく、「お店が間違えた」と本気で思い込んでいた。Vol.1のカイト（隠した）とは異なる動機タイプで差別化。「思い込み→黙る」の流れが教育的。
- **「2つのしなかった」がエピローグの構造的柱**: ①おつりを数えなかった ②財布を空にしたか確認しなかった。この2つを先生が明示することで、行動指針が明確に伝わる。
- **レジ記録による「お店は正しい」の早期確定**: Card2でお店の正しさを確定し、「10円は外部から」に焦点を絞る。推理の方向性が明確で、L1.5の難易度に適切。

## money-03（お金の探偵団 Vol.3）

**変換日**: 2026-03-25 / シーン12 / キャラ5 / 選択肢3問 / yuta→boy-b, sakura→girl-a, kaito→boy-d, nanoha→girl-c, sensei→teacher-m

### 気づき
- **二重割引の段階計算がVNの1クリック1ステップ構造に最適**: 「800×0.5＝400」→「400×0.8＝320」を2つのnarrationで出すと、「足し算じゃなくてかけ算の連鎖なんだ」とプレイヤーが自力で理解できる。算数の学習体験としてVNテンポが効果的。
- **「2つの間違いの偶然の一致」が推理の快感を生む**: ユウタの240円が「計算ミスで出した答え」と「本物の定価で正しく計算した答え」の偶然の一致。ナノハのセリフで段階的に明かすと「おお！」という驚きがある。数学的に面白い構造。
- **犯人がユウタ（情報型）に切り替わった効果**: Vol.1はカイト、Vol.2はナノハ、Vol.3でユウタが犯人に。「計算が得意」なキャラが計算ミス+定価改ざんという二重の問題を起こすギャップが面白い。
- **二重価格表示の社会的学び**: Card5で「本当のお店でも同じトリックがある」と接続。ネットのセールの「90%オフ！」を疑う力につながる。金融教育としての実用性が高い。
- **q2が計算教育になっている**: 「50+20=70%（足し算）」「段階的にかけ算（因果連鎖）」「50%×20%=10%（かけ算の間違い）」「分からない」——選択肢自体が誤答パターンの学習になる。
